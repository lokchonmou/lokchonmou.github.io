Question 2level 1

天氣經常陰晴不定，你需要做一個裝置：​

  1. 預設顯示今日天氣
  2. 按下按鍵1，顯示明日天氣
  3. 再按下按鍵1，顯示後日天氣，如此類推，直到未來7日天氣
  4. 按下按鍵2，回到上一日天氣
  5. 天氣資訊顯示在OLED上，內容自定，提交時請跟老師說明你選擇顯示了哪些資訊

Question 3level 2

慈幼中學即將迎來120周年，120年，每年365日，共有43,800日

學校現想發起挑戰，全校師生共同挑戰完成43,800次掌上壓

你需要製作多個相同功能的裝置：

  1. 師生在做掌上壓時，先要按下胸口的按鍵1，再按下背部的按鍵2，才算完成一次掌上壓
  2. 每完成一次掌上壓，顯示目前總完成次數
  3. 完成挑戰後(盡能力完成即到不能再做為止)，按下按鍵3可上傳資訊到雲端
  4. 你可以用OLED/web介面/Serial顯示裝置資訊，雲端資訊暫不用處理
  5. Bouns(level 3)根據雲端資料，製作一個實時折線圖，顯示累計師生掌上壓次數，並將43,800這個目標，用虛線橫線標示出來
