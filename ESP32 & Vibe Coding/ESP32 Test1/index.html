Question 3level 2

慈幼中學每年都會舉辦校運會，但校運會，但因技術所限，每年短跑、長跑均要人手計時，難以達到有效精準

現你需要製作很多個相同的裝置，先解決長跑問題：

  1. 當裁判按下按鈕，開始計時(按鍵1)
  2. 當運動員經過起點時，紀錄圈速，開始下一圈計時(按鍵2)
  3. 當運動員完成5000米(每圈400米)(13次trigger)，停止計時，並顯示最佳圈速和總完成時間
  4. 你可以用OLED/web介面/Serial顯示裝置資訊，雲端資訊和得獎者排序等暫不用處理
  5. 時間即時上傳至雲端，並附有"運動員編號(aka裝置編號)"、"最佳圈速"和"總完成時間"
  6. Bouns(level 3)利用雲端資訊，製作web介面即時顯示運動員最新排名和成績
